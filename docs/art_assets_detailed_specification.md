## 十、技术规格汇总

### 颜色空间
- **sRGB** 颜色空间
- 支持透明度 (Alpha通道)

### 动画帧率
- 游戏内动画: **12 FPS**
- UI呼吸动画: **8 FPS**
- 粒子动画: **12-15 FPS**

### 文件命名规范
```
[类型]_[名称]_[状态/变体].png

示例:
- snake_head_normal.png      (蛇头_正常)
- food_special_ghost.png     (食物_特殊_幽灵)
- terrain_forest_01.png      (地形_森林_变体1)
- ui_button_hover.png        (UI_按钮_悬停)
```

```
assets/

├── food/           # 食物素材 (11个)
├── terrain/        # 地形素材 (13个)

├── effects/        # 特效素材 (5个)

└── background/     # 背景素材 (3个)
```
