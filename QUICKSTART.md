# 🐍 Snake Game - 快速启动指南

## 🚀 快速开始

### 选项 1：使用原版（已验证稳定）

```bash
# 1. 在 Godot 中打开项目
# File -> Import -> 选择 G:\autoclawcode\project.godot

# 2. 按 F5 运行
```

### 选项 2：使用优化版（模块化架构）

```powershell
# 在 PowerShell 中运行

# 1. 备份原版
Copy-Item "G:\autoclawcode\scripts\Main.gd" "G:\autoclawcode\scripts\Main_original.gd"

# 2. 使用优化版
Copy-Item "G:\autoclawcode\scripts\Main_new.gd" "G:\autoclawcode\scripts\Main.gd"

# 3. 在 Godot 中重新加载并按 F5 运行
```

### 选项 3：回滚到原版

```powershell
# 如果优化版有问题，可以随时回滚

# 恢复原版
Copy-Item "G:\autoclawcode\scripts\Main_original.gd" "G:\autoclawcode\scripts\Main.gd"

# 在 Godot 中重新加载并按 F5 运行
```
